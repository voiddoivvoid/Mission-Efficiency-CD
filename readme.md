-Custom Mission Efficiency compatible with game version 1.07.00

-Ended support for Custom Mission Efficiency Generator (please post a comment on nexus if you want something custom, haven't tested in-game but theoretical max for a skill bonus before the calculation breaks and reverts to a smaller reflected bonus is 3000%. That number reflects a safe value where the base value of camp supplies received from a mission is 720. Each mission where the base value of camp supplies received grows higher than 720 the max skill bonus before things start breaking shrinks from that 3000% value)

-Removed 50x and 100x variants as these dont work. The game seems to be calculating more than a signed 32 bit integer.

-I will be adding a 0.5 and .75 times variant for those looking for more of a challenge and grind.
